# 🚖 OLA Ride Booking Data Analysis | SQL + Power BI

## 📌 Project Overview

This project is an end-to-end Data Analytics case study focused on OLA ride booking data. The objective is to analyze booking patterns, customer behavior, ride cancellations, revenue, ratings, and operational performance using SQL and Power BI.

The project demonstrates the complete analytics workflow, including data exploration, SQL-based business query solving, dashboard development, KPI tracking, and business insight generation.

---

## 📊 Project Objectives

• Analyze ride booking trends over time.
• Evaluate booking success and cancellation patterns.
• Identify revenue-driving vehicle categories.
• Measure customer and driver satisfaction.
• Analyze payment methods and customer spending.
• Build an interactive dashboard for business decision-making.

---

## 📂 Dataset

The dataset contains more than **100,000 OLA ride booking records** with attributes such as:

- Booking ID
- Date & Time
- Booking Status
- Customer ID
- Vehicle Type
- Pickup Location
- Drop Location
- Ride Distance
- Booking Value
- Payment Method
- Driver Rating
- Customer Rating
- Cancellation Reason
- Incomplete Ride Reason

---

## 🔍 SQL Analysis

The project answers important business questions using SQL, including:

- Retrieve successful bookings
- Average ride distance by vehicle type
- Customer cancellation analysis
- Driver cancellation analysis
- Top customers by bookings
- Revenue generated from completed rides
- Average customer ratings
- Driver rating analysis
- UPI payment analysis
- Incomplete ride analysis

Views were created for reusable reporting and simplified querying.
